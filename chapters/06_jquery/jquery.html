Build Your Own jQuery

Chapter 6 — DOM traversal, manipulation, and method chaining from scratch.

This entire page is driven by a 100-line $ function you built yourself. Every tab switch, accordion toggle, modal open, and card filter calls methods on wrapped element sets — with full chaining just like real jQuery. Inspect the source to see how.

What you built

Selector engine▼
$('selector') runs querySelectorAll and wraps the results in an object with chainable methods. Pass an Element directly and it wraps that too.
Method chaining▼
Every method that changes the DOM returns this, so you can chain: $('.btn').addClass('active').css('color', 'red').on('click', fn).
Event delegation▼
$.delegate(parent, event, selector, fn) attaches one listener on a parent and checks closest(selector) on each event — efficient for dynamic lists.
DOM manipulation▼
.html(), .text(), .append(), .prepend(), .remove(), .empty() — the same API surface as jQuery, built from raw browser APIs.
